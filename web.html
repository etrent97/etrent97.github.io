This page is here to educate visitors about web security:

Web security is the act of protecting websites from attacks such as unauthorized access or even website disruption. Website design principles require effort in a designing a secure website for users. There are many forms of website attacks that can completely compromise the website and any of the users personal information of whatever the attacker was going for. There are many safe practices to go by when building a website to ensure it has web security in place to avoid malicious attacks.
